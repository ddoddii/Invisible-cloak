## How to run 

### 1. Clone this repository

```shell
git clone https://github.com/ddoddii/Invisible-cloak.git
```

### 2. Install requirements

```shell
pip install -r requirements.txt
```

### 3. Run main.py

```shell
python3 main.py
```
